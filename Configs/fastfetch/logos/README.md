# Fastfetch ASCII logos

`arch-original.txt` was restored from `backups/fastfetch-20260514-133407.tar.gz`.

Extra Arch-based logos in the same ANSI/truecolor terminal style:

- `cachyos.txt`
- `endeavouros.txt`
- `manjaro.txt`
- `garuda.txt`
- `arcolinux.txt`
- `artix.txt`
- `rebornos.txt`

To switch logos, edit `Configs/fastfetch/config.jsonc` and change:

```jsonc
"source": "$HOME/.config/fastfetch/logos/arch-original.txt"
```

to one of the files above.
